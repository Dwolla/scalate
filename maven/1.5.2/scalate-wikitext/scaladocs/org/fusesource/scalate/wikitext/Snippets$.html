  6. def addPrefix (prefix: String, location: String): Unit

    Add a prefix definition to make it easier to access snippets.

    Add a prefix definition to make it easier to access snippets. E.g. if you add 'activemq' -> 'http://svn.apache.org/repos/asf/activemq/trunk' you can use 'activemq/pom.xml' as a shorthand notation for 'http://svn.apache.org/repos/asf/activemq/trunk/pom.xml"

    prefix

    the prefix

    location

    the full URL that should be used for the prefix
